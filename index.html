JUG'S HTML5 Demo

About

This is a simple demo application for HTML5 features for mobile plattforms and for JQueryMobile. The demo was presented as part of a talk by Corsin Decurtins from Netcetera for the Java User Group Switzerland.

The source code for the application is available on GitHub.
